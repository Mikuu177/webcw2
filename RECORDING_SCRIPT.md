# WebCW2 Recording Script

This script is designed for a 5-minute video. Follow it in order. Each command has a matching sentence to say immediately after the output appears.

## Before Recording

Open a terminal and run:

```bash
cd /Volumes/Mikuu-ultra/Year4/web/webcw2
source .venv/bin/activate
```

Open these files in your editor:

- `README.md`
- `src/crawler.py`
- `src/indexer.py`
- `src/search.py`
- `GENAI_REFLECTION.md`

Use a large terminal font so the text is readable in the video.

Screen: open `src/crawler.py`.

Say:

```text
The data flow starts in crawler.py. It fetches quote pages, extracts quote text, follows the next-page link, and respects the required politeness window.
```

```text
The sleeper is injectable, which means the real program waits six seconds, while tests can mock that delay and still verify the behavior.
```

Screen: open `src/indexer.py`.

Say:

```text
Next, indexer.py normalizes text to lowercase, removes punctuation, and builds the inverted index.
```

Point at `TOKEN_RE`, `Posting`, and `add_page`.

Say:

```text
Each term maps to each page with frequency and positions. This satisfies the requirement to store word statistics, not only page URLs.
```

Screen: open `src/search.py`.

Say:

```text
Finally, search.py implements print and find. Multi-word queries are handled by intersecting page sets, then results are ranked.
```

Point at `find`, `_tf_idf`, and `suggest`.

Say:

```text
TF-IDF rewards repeated matches on a page and gives more weight to distinctive terms. The suggestion helper improves usability for misspelled or missing words.
```
